# Система оценки — Audience Lens

---

## Уровень 1: Оценка аватарами

### 4 метрики (0-100 каждая)

| Метрика | Вес | Что измеряет |
|---------|-----|-------------|
| **Relevance** | 0.30 | Про меня ли этот контент? Про мою ситуацию, боль, вопрос? |
| **Emotional** | 0.25 | Эмоциональный отклик: узнавание, облегчение, мотивация, страх |
| **Useful** | 0.25 | Могу применить? Получил новое знание/инструмент? |
| **Shareable** | 0.20 | Сохраню? Перешлю? Прокомментирую? |

### Формула

```
Avatar Score = Relevance × 0.30 + Emotional × 0.25 + Useful × 0.25 + Shareable × 0.20
```

### Шкала реакций

| Score | Эмодзи | Реакция | Что делает аватар |
|-------|--------|---------|-------------------|
| 85-100% | 🔥 | Very Positive | Лайк + сохранение + комментарий + может репост |
| 70-84% | 👍 | Positive | Лайк, возможно сохранение |
| 50-69% | 😐 | Neutral | Прочитал, пролистнул |
| 30-49% | 👎 | Negative | Не дочитал, лёгкое раздражение |
| 0-29% | ⏭️ | Skip | Пролистал не читая |

## Уровень 2: Оценка критиками

### 3 критика (1-100 каждый)

| Критик | Что оценивает | Ключевые вопросы |
|--------|--------------|-----------------|
| **Sales** | Продающая сила | Есть ли CTA? Понятен ли следующий шаг? Работают ли триггеры? Ведёт ли в воронку? |
| **Creative** | Творческая сила | Цепляет ли хук? Свежая ли подача? Есть ли эмоция? Запомнится ли? |
| **Structure** | Структурная ясность | Логична ли последовательность? Лёгок ли для чтения? Правильный ли объём? |

### Шкала критиков

| Score | Уровень | Значение |
|-------|---------|----------|
| 85-100 | Отлично | Образцовый по этому параметру |
| 70-84 | Хорошо | Работает, мелкие улучшения |
| 50-69 | Средне | Заметные слабости |
| 30-49 | Слабо | Серьёзные проблемы |
| 0-29 | Плохо | Не выполняет функцию |

## Комбинированный вердикт

### Матрица решений

```
✅ PUBLISH:
  Аватары ≥70% И Критики ≥70
  ИЛИ хотя бы один аватар ≥85% И Критики ≥60 (нишевый)

⚡ REVISE:
  Аватары 50-69% И Критики ≥50
  ИЛИ Аватары ≥70% И Критики 50-69

❌ REWRITE:
  Аватары <50%
  ИЛИ Критики <50
  ИЛИ любой аватар <20% с негативной критикой (отталкивает часть аудитории)
```

### Тип контента

| Паттерн | Тип | Действие |
|---------|-----|----------|
| Все аватары 70%+ | **Массовый** | Широкий охват, ставить в прайм-тайм |
| 1-2 аватара 85%+, остальные <50% | **Нишевый** | Пометить тегом сегмента, может ставить не в прайм |
| Разброс >50 пунктов между аватарами | **Поляризующий** | Проверить — не отталкивает ли часть ЦА |
| Все аватары 50-69% | **Серый** | Никого не задевает — самый плохой вариант. Переписать |

### Финальный скор

```
Thread Score = Среднее всех аватаров
Critics Score = Среднее 3 критиков
Combined = Thread Score × 0.6 + Critics Score × 0.4
```

Вес аватаров выше (0.6), потому что резонанс с аудиторией важнее формального качества.

## Особые случаи

### "Серый" контент (все 50-69%)
Самый опасный результат. Контент не раздражает, но и не цепляет. Алгоритмы его хоронят, аудитория забывает.
**Действие:** Переписать с усилением для 2-3 аватаров. Лучше нишевый 🔥 для одного, чем 😐 для всех.

### Негативная реакция (аватар <20% с критикой)
Если хоть один аватар даёт <20% И объясняет почему — это сигнал.
**Действие:** Проверить, не оттолкнёт ли контент часть реальной аудитории. Если аватар из ядра ЦА — обязательная доработка.

### Один аватар 🔥, остальные ⏭️
Нормально для экспертного контента. Публиковать как нишевый.
**Но:** Если 80% контента нишевый — нет массовых постов для охвата. Нужен баланс.
