# Шаблон аватара для Audience Lens

> Каждый аватар — это конкретный человек из ЦА. Не абстракция, а живой персонаж с болью, контекстом и поведением.

---

## Шаблон

```markdown
### Аватар: {Имя} ({возраст} лет)

**Кто:** {профессия/роль}, {семейное положение если релевантно}

**СИТУАЦИЯ:**
- {Главная боль — конкретная, не абстрактная}
- {Что уже пробовал(а) — и не помогло}
- {Что случилось недавно — триггер, последняя капля}
- {Чего хочет на самом деле — глубинное желание}

**ЧТО ЦЕПЛЯЕТ В КОНТЕНТЕ:**
- {Тема 1 — конкретный тип поста}
- {Тема 2}
- {Тема 3}
- {Формат — что нравится: истории, списки, объяснения, кейсы}

**ЧТО ОТТАЛКИВАЕТ:**
- {Раздражитель 1 — конкретный}
- {Раздражитель 2}
- {Раздражитель 3}

**КАК РЕАГИРУЕТ НА КОНТЕНТ:**
- Если узнаёт себя → {действие: лайк, сохранение, комментарий, репост}
- Если полезно, но не про неё/него → {действие}
- Если раздражает → {действие: пролистывает, отписка}

**ГОЛОС В КОММЕНТАРИЯХ:**
- "{типичная фраза 1}"
- "{типичная фраза 2}"
- "{типичная фраза 3}"
```

## Правила создания аватаров

### 1. Конкретность > обобщения
```
❌ "Женщина 25-35 лет, интересуется саморазвитием"
✅ "Катя, 31 год, HR-менеджер. Выгорела после 3 лет на одном месте, 
    читает про смену профессии, но боится потерять доход."
```

### 2. Боль должна быть острой
```
❌ "Хочет больше зарабатывать"
✅ "Зарабатывает 120к, ипотека 45к, на отпуск не остаётся. 
    Видит в ленте людей с фриланса, бесит что они путешествуют."
```

### 3. Разные сегменты — разные аватары
- Не делай 10 похожих людей
- Каждый аватар представляет ОТДЕЛЬНЫЙ сегмент или подсегмент
- Разные боли, разные триггеры, разные стадии осознания проблемы

### 4. Стадия осознания проблемы
Отметь для каждого аватара:
- 🟢 **Осознал проблему** — ищет решение
- 🟡 **Подозревает** — чувствует дискомфорт, не формулирует
- 🔴 **Не осознал** — думает что "всё нормально", нужно зеркало

### 5. Источники для создания
- Кастдев-интервью (customer-research)
- Комментарии в соцсетях конкурентов
- Реальные вопросы из директа/чата
- Отзывы на продукт эксперта
- Данные анкет предзаписи

## Сколько аватаров нужно

| Количество | Оценка |
|------------|--------|
| 1-4 | ❌ Слишком мало — перекос в сторону одного сегмента |
| 5-7 | ✅ Минимум — покрывает основные сегменты |
| 8-10 | ✅✅ Оптимально — полная картина аудитории |
| 11+ | ⚠️ Избыточно — оценка размывается, сложно обрабатывать |

### Распределение по сегментам
Если 3 основных сегмента ЦА:
- Сегмент 1 (ядро, 50% аудитории): 3-4 аватара
- Сегмент 2 (важный, 30%): 2-3 аватара
- Сегмент 3 (периферия, 20%): 1-2 аватара

## Пример: ниша "онлайн-курсы по маркетингу"

```markdown
### Аватар: Лена (29 лет)

**Кто:** SMM-менеджер в агентстве, не замужем

**СИТУАЦИЯ:**
- Застряла на 80к зарплаты, потолок в агентстве
- Прошла 3 курса по таргету — знания есть, клиентов нет
- Последний фриланс-клиент не заплатил — 2 недели работы впустую
- Хочет выйти на 200к+ через свои проекты, но не знает как продавать себя

**ЧТО ЦЕПЛЯЕТ В КОНТЕНТЕ:**
- Кейсы "как я ушёл из найма в 200к на фрилансе"
- Конкретные инструменты (шаблоны КП, скрипты переговоров)
- Истории провалов с выводами

**ЧТО ОТТАЛКИВАЕТ:**
- "Я заработал миллион за первый месяц" — не верит
- Теория без практики
- Посты только для предпринимателей (она пока наёмник)

**КАК РЕАГИРУЕТ НА КОНТЕНТ:**
- Если узнаёт себя → сохраняет, пересылает подруге-коллеге
- Если полезно, но не про неё → лайк, пролистывает
- Если раздражает → закатывает глаза, пролистывает

**ГОЛОС В КОММЕНТАРИЯХ:**
- "О, мне как раз это нужно было!"
- "А это работает для SMM или только для таргета?"
- "Сохранила, спасибо 🙏"
```
