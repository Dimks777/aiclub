# Reels/Shorts/TikTok — Система вирусного контента

Полный цикл от идеи до продаж. Не просто "как снять рилс", а как построить машину продаж через вертикальный контент.

**Философия:** Reels — это НЕ контент. Reels — это верхушка автоматизированной воронки продаж.

**Формула успеха:** Первые 3 секунды = 80% успеха ролика.

## Формула ВИСП (первые 3 секунды)

Каждый опенинг должен закрывать минимум 2-3 элемента:

| Элемент | Суть | Проверочный вопрос |
|---------|------|--------------------|
| **В**ыгода | Что зритель получит | Это желание 5-летнего? |
| **И**нтрига | То, чего он не знает | Ответ заменён на "это"? |
| **С**рочность | Почему важно сейчас | Есть повелительное наклонение? |
| **П**ричастность | Узнавание себя | Зритель узнаёт себя? |

**Техника "Дед Мороз":** Если бы 5-летний просил это у Деда Мороза — выгода хорошая.

**"200 миллионов":** Тема понятна каждому человеку на планете.

## Структура сценария

```
Проблема → Актуализация → (Самопрезентация) → Решение → Вывод → CTA
```

Тайминги (золотой стандарт 60 сек):
```
[0-3 сек]  ХУК по ВИСП
[3-10 сек] АКТУАЛИЗАЦИЯ — почему важно ТЕБЕ
[10-12 сек] САМОПРЕЗЕНТАЦИЯ (опционально, кратко!)
[12-45 сек] РЕШЕНИЕ с возвратами внимания каждые 10-15 сек
[45-55 сек] ВЫВОД
[55-60 сек] CTA
```

## Воронка продаж (экспресс-формула)

```
REELS → Кодовое слово → Instagram бот → Telegram → Лид-магнит → КЭВ → Продажа
10K      3% (300)        80% (240)       50% (120)   25% (30)    35% = 10-15 продаж
```

При среднем чеке 100K₽ = 1-1.5 млн₽. Система работает 24/7.

**CTA по ВИСП, не "переходи по ссылке":**
- ✅ "Я собрал чек-лист из 15 ошибок — напиши ПЛАН в директ, отправлю бесплатно"
- ❌ "Переходи по ссылке в шапке профиля"

## Правила устной речи в скриптах

Скрипт — это НЕ статья. Это устная речь.

1. **Прочитай вслух.** Язык спотыкается — переписывай.
2. **Добавь паузы.** Многоточие, обрыв мысли — норма для устной речи.
3. **Не пиши как для текста.** "Осуществление данного процесса" → "Делаешь вот так".
4. **Варьируй ритм.** Короткая фраза. Потом длинная с деталями. Снова короткая.
5. **Тест "голосовое сообщение".** Если не звучит как голосовое другу — это не скрипт.

## Диагностика проблем

| Симптом | Причина | Решение |
|---------|---------|---------|
| Мало просмотров | Слабые первые 3 сек | ВИСП-опенинг |
| Подписок нет | Нет CTA | "Здесь всё о..." |
| Продаж нет | Нет воронки | Кодовое слово → бот |
| Низкий retention | Ответ дан сразу | Техника "Печёночный торт" |

## Ключевые техники

- **"Печёночный торт"** — ответ в самом конце ролика
- **"Дед Мороз"** — выгода как желание 5-летнего
- **"200 миллионов"** — понятно каждому человеку на планете
- **RPG-подход** — каждое улучшение качества = +N к привлечению

## Рилс при запуске продукта

Рилсы во время конверсионных этапов (анкета, регистрация на вебинар) = **+20% к заполнениям**.

**Формула рилса при запуске:**
```
Боль/потребность/возражение → Закрытие кейсом/отсылкой к продукту
```

- Несколько рилсов в неделю во время прогрева
- Связывать логически: Пост + Сторис + Рилс
- При низких охватах в сторис — рилсы/посты набирают больше

## AI Voiceover (HeyGen и другие)

При использовании AI-аватара текст надо адаптировать:

| Правило | ❌ Не делать | ✅ Делать |
|---------|-------------|----------|
| **Кавычки** | "токсичный" продукт | так называемый токсичный продукт |
| **Числа** | 50 лет, 3 раза | пятьдесят лет, три раза |
| **Стаккато** | Боль. Усталость. Слабость. | Боль, усталость и слабость |
| **Гипофора** | Что делать? Вот что. | Решение простое: делать это |
| **Длина** | Предложения 30+ слов | Предложения до 20 слов |

**Гипофора — главный враг AI-озвучки:** убрать вопрос, начать сразу с ответа.

## Частые ошибки

❌ **Ответ сразу в опенинге** — главный убийца удержания
❌ **Несколько тем в одном ролике** — 1 видео = 1 проблема
❌ **"Топ-5 лучших"** — заменять на "3 самых...", "5 [чего-то]..."
❌ **Развлекательный контент без экспертности** — нецелевая аудитория
❌ **CTA "переходи по ссылке"** — слишком прямолинейно
❌ **Игнорирование первых 3 секунд** — 80% успеха решается здесь
❌ **Однообразная картинка** — нужны субтитры и перебивки
❌ **Ежедневный постинг вместо качества** — раз в 2-3 дня лучше, чем треш каждый день
❌ **Копирование конкурентов 1:1** — брать формат, менять смыслы
