# Text Posts -- Ядро текстового контента

Универсальная система создания текстовых постов. Платформо-независимое ядро, которое используют скиллы threads, telegram и другие платформенные обертки.

**Принцип:** Сильный пост работает на любой платформе. Платформа определяет формат, не суть.

## Быстрый старт

```bash
# Пост по теме
"Напиши пост на тему: [ТЕМА]"

# Пост конкретного типа
"Напиши пост-инструмент про [ТЕМА]"

# Хуки по теме
"Сделай 5 хуков на тему: [ТЕМА]"

# Пакет постов
"5 постов на неделю для ниши [НИША]"
```

## File Router

| Файл | Содержание | Когда читать |
|------|-----------|--------------|
| `post-anatomy.md` | Анатомия поста, 6 типов контента, структуры, принципы написания, workflow, антипаттерны | Написание любого поста |
| `hooks-and-cta.md` | Формулы хуков, CTA-система, чеклист публикации | Работа с первой строкой, призывами к действию |

## Зависимости (читай при необходимости)

| Скилл | Когда подключать |
|-------|-----------------|
| **copywriting** | Фундамент: скользкая горка, 18 правил, работа с ценой и возражениями |
| **storytelling** | Когда пост = история (трехактная структура, психологические триггеры) |
| **selling-meanings** | Когда пост продает (распаковка смыслов, связка боль+кейс) |
| **headlines** | Когда нужны заголовки для длинных форматов |
| **shared-learnings** | Антипаттерны и тональные формулы -- проверка перед публикацией |

## Workflow создания поста

```
1. ТЕМА     -- о чем пост? Одна идея.
2. ТИП      -- инструмент / закулисье / личное / философия / тренд / продукт
3. ХУК      -- потрать 50% времени. 3-5 вариантов, выбери лучший
4. ТЕЛО     -- раскрой через структуру типа (см. post-anatomy.md)
5. ИНСАЙТ   -- вывод или мораль (что читатель унесет)
6. CTA      -- мягкий, средний или с вопросом (см. hooks-and-cta.md)
7. СОКРАЩАЙ -- убери 30%. Каждое предложение должно тянуть к следующему
8. ПРОВЕРКА -- чеклист из hooks-and-cta.md
```

## Платформенные обертки

После создания ядра поста, адаптируй под платформу:

| Платформа | Скилл | Что добавляет |
|-----------|-------|---------------|
| Threads | `threads` | Лимит 500, ветки, алгоритм, engagement |
| Telegram | `telegram` | 4096 символов, bold, воронки, закреп |
| Instagram | `carousel` / `reels` | Визуал, слайды, вертикальное видео |
