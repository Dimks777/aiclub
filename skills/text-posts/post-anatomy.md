# Post Anatomy -- Структуры и типы контента

## META

| Field | Value |
|-------|-------|
| **Module** | Post Structure |

| **Status** | Active |

## Анатомия поста (универсальная)

```
[ХУК]      -- Scroll-stopper. Решает, прочитают ли дальше
    |
[КОНТЕКСТ] -- 1-3 предложения: почему это важно, актуализация
    |
[ТЕЛО]     -- Раскрытие: списки, примеры, история, аргументы
    |
[ИНСАЙТ]   -- Вывод, мораль, главная мысль (можно в цитате)
    |
[CTA]      -- Призыв к действию (опционально)
```

**Золотое правило:** Задача каждого предложения -- заставить прочитать следующее. Это скользкая горка (подробнее: скилл copywriting).

## 6 типов контента

### 1. Инструмент (25-30%)

Обзор полезного инструмента, метода, приема.

**Структура:**
```
[Инструмент] -- [что делает / какую проблему решает]

[Контекст проблемы -- 1-2 предложения]

Что умеет / Что дает:
- [Функция/выгода 1]
- [Функция/выгода 2]
- [Функция/выгода 3]

Как использую:
[Личный опыт с конкретикой -- числа, время, результат]

[CTA -- опционально]
```

**Пример:**
> **SuperWhisper -- печатаю голосом быстрее, чем пальцами.**
>
> Раньше писал пост 40 минут. Теперь -- 10.
>
> Что умеет:
> - Распознает речь на лету
> - Авто-пунктуация
> - Работает оффлайн
>
> Использую каждый день для Telegram, заметок, сценариев.

---

### 2. Закулисье (20-25%)

Показ процесса, внутренней кухни, рабочих моментов.

**Структура:**
```
[Заголовок -- что делаю / что случилось]

[Контекст ситуации]

[Процесс / Что происходило]

[Инсайт / Что понял]

> [Вывод -- опционально]
```

**Пример:**
> **На моей фабрике контента сотрудник No1 -- ИИ-сценарист.**
>
> Раньше сценарии для YouTube писал сам. По 4-5 часов на ролик.
>
> Сейчас:
> - Голосовая заметка с идеей
> - Промпт в Claude
> - Редактура 30 минут
>
> Не заменил себя -- усилил. Думаю, не печатаю.

---

### 3. Личное (15-20%)

Личная история, уязвимость, путь.

**Структура:**
```
[Заголовок -- ситуация или эмоция]

[Что произошло]

[Как себя чувствовал / Что думал]

[Что понял / Как изменилось]
```

**Пример:**
> **Я пробежал 100 км за неделю и вот, что я понял.**
>
> Не планировал. Просто каждый день выходил.
>
> На третий день хотел бросить. Болели ноги, было лень.
>
> Но продолжил. И понял: дисциплина -- это не когда легко. Это когда продолжаешь, несмотря на.

---

### 4. Философия (10%)

Смелый тезис, позиция, мировоззрение.

**Структура:**
```
[Тезис -- смелое утверждение]

[Почему так думаю -- 2-3 аргумента]

[Личный пример как доказательство]

> [Мораль или призыв к размышлению]
```

**Пример:**
> **Залетевший ролик -- это не везение. Это статистика.**
>
> Когда выпускаешь 3 ролика в месяц -- надеешься на удачу.
> Когда 30 -- работает математика.
>
> У меня было 7 "залетов" за год. Все -- после серии из 10+ видео подряд.
>
> Количество создает качество. Не наоборот.

---

### 5. Тренд (10%)

Реакция на событие, тренд, новость в нише.

**Структура:**
```
[Тренд] -- [оценка или вопрос]

[Что это / Как работает -- кратко]

[Моя позиция -- за или против, с аргументами]

[Вопрос к аудитории -- опционально]
```

---

### 6. Продукт (5-10%)

Анонс, результат клиента, упоминание продукта.

**Структура:**
```
[Результат или анонс]

[Что это / Для кого -- кратко]

[Что внутри / Что получит]

[CTA -- мягкий]
```

**Важно:** Продуктовый контент = максимум 10%. Подробнее про продающие тексты -- скилл copywriting (формула "продавать не продавая").

## Структуры длинных постов

### Problem -> Solution -> Experience
```
[Проблема -- боль аудитории]
[Решение -- что помогает]
[Мой опыт -- как использую]
[Вывод]
```

### Story Arc (для историй)
```
[Завязка -- ситуация]
[Развитие -- что происходило]
[Кульминация -- поворот]
[Мораль -- что понял]
```

### List Format (для инструментов)
```
[Заголовок]
1. [Пункт]
2. [Пункт]
3. [Пункт]
[Личный опыт]
[CTA]
```

### Pain + Case (для продающих -- из copywriting)
```
[Боль/потребность/возражение]
[Раскрытие через историю/экспертность]
[Закрытие кейсом или отсылкой к продукту]
```

## Принципы написания

| Принцип | Как применять |
|---------|---------------|
| **Одна идея = один пост** | Не размывай. Если две мысли -- два поста |
| **Хук = 50% времени** | Первое предложение решает все |
| **Конкретика** | Числа, названия, примеры. Мозг фильтрует абстракции |
| **Личный опыт** | "Я сделал" > "Можно сделать" |
| **Короткие абзацы** | 2-3 предложения max |
| **Четкая позиция** | Не "может быть", а "я считаю". Быть за белых или за красных |
| **Разговорный стиль** | Как рассказываешь другу |
| **Скальпель** | Сократи на 30%. Нет лишних тезисов, примеров, оборотов |

## Тональность (универсальная база)

### Прямолинейность
- Короткие предложения
- Без "воды"
- Конкретика вместо абстракций

### Уязвимость
- Делиться неудачами
- Показывать процесс, не только результат
- "Я бросал", "Было сложно", "Не получилось"

### Честность
- И успехи, и провалы
- Не преувеличивать результаты

**Важно:** Это базовая тональность. Для персонализации используй скилл tone-of-voice (если настроен для проекта).

## Человечность текста (Anti-AI)

> ИИ пишет правильно. Человек пишет живо. Если текст звучит "грамотно, но пусто" -- проблема здесь.

Подробные техники: скилл storytelling (`kadyrov-techniques.md`). Здесь -- минимальный чеклист.

### 7 правил живого текста

1. **Конкретная сцена > обобщение.** Вместо "цените близких" -- история про конкретного деда в тренажёрке. Место, имя, число, физическое ощущение.

2. **Уязвимость без нытья.** "Мне хреново, но вот что я с этим делаю." Не просить жалости. Не обесценивать: "всё будет хорошо" -- мёртвая фраза.

3. **Самоподрыв серьёзности.** В пафосные моменты -- одна строка, которая сбивает градус. ИИ всегда нарастает к финалу. Человек может оборвать.

4. **Незавершённость.** Не каждый пост заканчивается бантиком. "Не знаю пока" -- нормальный финал. ИИ всегда завершает красиво -- человек заканчивает как заканчивает.

5. **Физические ощущения.** "Устал" -- мёртвое слово. "Морально высушен" -- ты чувствуешь. Один физический образ > десять прилагательных.

6. **Диалоги.** Живые реплики в тексте мгновенно оживляют любой пост. Короткие, рваные, не литературные.

7. **Поток, не статья.** Длинный пост = поток сознания с внутренней драматургией. Структура скрыта, не торчит наружу подзаголовками.

### Шкала убедительности

```
абстракция < красивые слова < конкретные детали < физические ощущения < диалог < пруф
```

## Anti-patterns

| Ошибка | Почему плохо | Как исправить |
|--------|--------------|---------------|
| Длинное вступление | Теряется внимание | Сразу в суть |
| Generic advice без примера | Не запоминается | Добавь личный опыт |
| "AI" вместо "ИИ" | Не естественно для русского текста | Использовать "ИИ" |
| Стрелки -> в тексте | Маркер нейросети | Заменить на тире или точку |
| "Серьезно." как усилитель | Паттерн ChatGPT | Убрать или переформулировать |
| "Это не про X, это про Y" | Маркер ИИ-текста | Писать прямо, без шаблонных противопоставлений |
| Канцеляризмы | Мертвый язык | "Зеленый массив" -> "лес" |
| Клише | Не работают | "Уютный уголок", "индивидуальный подход" -- убрать |
| Больше 3 коммерческих слов | Давление на читателя | Мягче, через ценность |
| Слова инфобизнеса | Мертвый язык экспертов | "масштабирование" -> "вырасти", "точка Б" -> убрать |
| Красивый финал всегда | Предсказуемо | Иногда оборвать, когда мысль закончилась |
| Обобщение вместо сцены | Не запоминается | Конкретная история с именами и местами |
| Нейтральный тон | Скучно | Каждый пост должен быть эмоционально заряжен |
