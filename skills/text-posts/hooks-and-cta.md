## Почему хук решает все

- 1-2 секунды на захват внимания
- Первая строка видна в ленте -- решает scroll-stop или scroll-past
- Время удержания -- ключевая метрика алгоритмов
- **Правило 50%:** потрать половину времени на хук

## 5 базовых триггеров

| Триггер | Описание | Пример |
|---------|----------|--------|
| **Зависть** | Результат, которого хочет ЦА | "Как я набрала 11к подписчиков за месяц" |
| **Жадность** | Выгода, польза, экономия | "5 скрытых функций ChatGPT, о которых молчат 99%" |
| **Любопытство** | Интрига, незавершенность | "Все говорят про X, но я увидела другую закономерность..." |
| **Тайное знание** | Инсайдерская информация | "Почему 80% онлайн-курсов убивают мотивацию" |
| **Результат** | Конкретные цифры | "327 подписчиков за 5 дней с нуля" |

## 8 формул хуков

### 1. Контрарианство
```
[Общепринятое мнение] -- это неправда/миф/ошибка.
```
> "Количество убивает качество" -- это миф.

### 2. Исследование/Статистика
```
Исследование [авторитет] показало [шокирующий факт]...
```
> Исследование Гарварда уничтожило миф о деньгах и счастье.

### 3. Личная история с конфликтом
```
[Ситуация из прошлого] изменила все / научила меня...
```
> Клиентка отказалась работать со мной, оценив меня по кухне.

### 4. Цифры + Обещание
```
[X] способов/правил/ошибок, которые [желаемый результат]
```
> 5 приемов, которые делают снимки в 3 раза более вовлекающими.

### 5. Провокационный вопрос
```
Почему [целевая аудитория] до сих пор [ошибка]?
```
> Почему 99% фотографов снимают портреты неправильно?

### 6. Эмоциональный взрыв
```
[Эмоция]! [Событие + контекст]
```
> Первая продажа с Telegram! 3 недели работы над воронкой.

### 7. Интрига с обрывом
```
[Начало истории]... и тогда все изменилось.
```
> 3 недели назад принял решение, которое изменило все.

### 8. Экспертная декларация
```
Как [профессия] официально заявляю: [смелое утверждение]
```
> Как психолог официально заявляю: вы не сломаны.

## Хуки по типу контента

### Для инструментов
| Формула | Пример |
|---------|--------|
| **[Инструмент] -- [что дает]** | SuperWhisper -- печатаю голосом быстрее пальцев |
| **[Проблема] -> [Решение]** | Час на пост? С этим -- 10 минут |
| **[Результат с инструментом]** | Сценарий за 30 минут вместо 4 часов |

### Для закулисья
| Формула | Пример |
|---------|--------|
| **На моей [X] работает [Y]** | На моей фабрике контента сотрудник No1 -- ИИ |
| **[Процесс] -- как это устроено** | Как я собираю контент на неделю |
| **[Время] назад я [действие]** | Месяц назад перестроил всю систему |

### Для личного
| Формула | Пример |
|---------|--------|
| **[Действие] + инсайт** | Пробежал 100 км за неделю -- вот что понял |
| **[Ситуация] научила меня** | Провал запуска научил меня главному |
| **[Состояние]** | Устал. Но продолжаю. |

### Для философии
| Формула | Пример |
|---------|--------|
| **[Тезис]** | Количество создает качество. Не наоборот. |
| **[Контрарианство]** | Все говорят про X -- я вижу другое |
| **[Вопрос + позиция]** | Почему мы боимся выкладывать "сырое"? |

### Для продукта
| Формула | Пример |
|---------|--------|
| **[Результат клиента]** | Ученик сделал первые 100k -- разбираю |
| **[Анонс + ценность]** | В субботу -- эфир про воронки. Без воды. |
| **[Новость + польза]** | Обновил шаблон Notion -- забирай |

### Для вопросов (вовлечение)
| Формула | Пример |
|---------|--------|
| **[ЦА], расскажите, как вы [действие]?** | Психологи, расскажите, как вы пришли в профессию? |
| **[Интересный факт]. А как у вас?** | Я узнала, что 90% людей... А вы? |

## Антипаттерны хуков

| Плохой хук | Почему | Как исправить |
|------------|--------|---------------|
| "Сегодня хочу поговорить о..." | Скучно, нет интриги | Начни с результата или проблемы |
| "Всем привет! Как дела?" | Пустой, не цепляет | Сразу в суть с триггером |
| "Кайф." (одно слово) | Нет контекста | Добавить конкретику: "Кайф, когда план работает" |
| "Серьезно." | Паттерн нейросети | Переформулировать |
| "Вы знали, что..." | Избитый формат | Личный опыт вместо "вы" |
| Агрессивный вопрос | Отталкивает | "Почему у тебя нет денег?" -> "Какие денежные привычки работают?" |
| Кликбейт без payoff | Теряется доверие | Хук должен соответствовать содержанию |

## CTA-система

### 3 типа CTA

| Тип | Описание | Примеры |
|-----|----------|---------|
| **Мягкий** | Упоминание как факт, не продажа | "В Фабрике показывал...", "В субботу на эфире разберу" |
| **Средний** | Прямое действие | "Пиши в личку [слово]", "Ссылка в описании" |
| **С вопросом** | Вовлечение через диалог | "Как у вас с этим?", "Что думаете?" |

### 3 типа по действию (из copywriting)

| Тип | Примеры |
|-----|---------|
| **Коммуникационный** | Лайк, комментарий, репост, сохранение |
| **Продающий** | Ссылка, заявка, консультация, регистрация |
| **Эмоциональный** | Запомните, улыбнитесь, начните действовать |

### Правила CTA

1. **Максимум 5 CTA на текст.** Не повторять один и тот же
2. **Формула:** Действие + мотив для читателя
3. **Пример:** "Сохраните пост, чтобы сервисы были под рукой"
4. **Продуктовый CTA** -- только мягкий, кроме продуктовых постов

### 3 точки размещения CTA

1. **После вводной** -- "Сохраните пост" (до основной части)
2. **В контенте** -- короткие вставки по ходу текста
3. **В конце** -- ключевой призыв после всей ценности

## Чеклист перед публикацией

### Критические (блокируют публикацию)
- [ ] Хук содержит конкретику (не просто эмоция)?
- [ ] Нет длинного вступления?
- [ ] Есть личный опыт/пример?
- [ ] Одна идея на пост?
- [ ] Четкая позиция (не "с одной стороны, с другой")?
- [ ] Нет маркеров ИИ-текста (стрелки, "серьезно", "это не про X, это про Y")?
- [ ] "ИИ" вместо "AI"?

### Качество
- [ ] Пост можно пересказать за 30 секунд?
- [ ] Каждое предложение тянет к следующему (скользкая горка)?
- [ ] CTA отдельным блоком (если есть)?
- [ ] Нет канцеляризмов и клише?
- [ ] Прочитал вслух -- не спотыкается?

### Перед адаптацией под платформу
- [ ] Проверил длину (подходит для целевой платформы)?
- [ ] Учел ограничения платформы (ссылки, форматирование)?
- [ ] Применил платформенную обертку (threads/telegram/etc)?

## Workflow: от идеи к посту

```
1. Напиши 3-5 вариантов хука
2. Выбери лучший (триггер + конкретика)
3. Определи тип контента
4. Напиши тело по структуре типа (post-anatomy.md)
5. Добавь личный опыт
6. Сформулируй инсайт/вывод
7. Добавь CTA (если уместно)
8. Сократи на 30%
9. Проверь по чеклисту
10. Адаптируй под платформу
```
