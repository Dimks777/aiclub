## Как пользоваться

1. Создай ядро поста по скиллу **text-posts** (структура, хук, CTA)
2. Адаптируй под Threads по правилам ниже

## Технические ограничения

| Параметр | Значение |
|----------|----------|
| Символов в ветке | 500 max |
| Количество веток | Неограничено |
| Видео | До 5 минут |
| Картинки | Да, повышают конверсию |
| Ссылки в посте | Режутся алгоритмами! |

## Анатомия поста в Threads

```
[ХУК]      -- Первая строка, scroll-stopper
    |
[ИНТРИГА]  -- Зацепка для перехода в ветку
    |
[ТЕЛО]     -- 1-10 веток по 500 символов
    |
[CTA]      -- Призыв к действию
    |
[ССЫЛКА]   -- ТОЛЬКО в закрепленном комментарии!
```

**Ключевое отличие от text-posts:** все через ветки. Длинный пост = несколько веток по 500 символов.

## Структуры через ветки

### PAS (Problem-Agitate-Solve)
```
Ветка 1: [Проблема] -- боль аудитории
Ветка 2: [Усиление] -- что будет, если не решить
Ветка 3-4: [Решение] -- ваш метод
Ветка 5: [CTA] -> закреп
```

### Story Arc
```
Ветка 1: [Завязка] -- ситуация + интрига
Ветка 2: [Развитие] -- что произошло
Ветка 3: [Кульминация] -- поворот
Ветка 4: [Развязка] -- чем закончилось
Ветка 5: [Мораль + CTA]
```

### List Format
```
Ветка 1: [Хук] X способов/ошибок/правил
Ветка 2: 1-2 пункта
Ветка 3: 3-4 пункта
Ветка 4: 5+ пункта
Ветка 5: [CTA] -> закреп
```

## Threads-специфичные типы контента

Помимо 6 универсальных типов из text-posts, в Threads работают:

### Вопросы (для раскачки аккаунта)
```
[Контекст/Пример]
[Вопрос, на который ХОЧЕТСЯ ответить]
```
- "Психологи, расскажите, как вы пришли в профессию?"
- "Какие у вас денежные ритуалы?"
- **Важно:** НЕ агрессивные вопросы!

### Лайк-посты / Пожелания
```
[Пожелание/Утверждение]
Лайк, если [согласен/хочешь]
```
- "Самый приятный звук -- уведомление о поступлении денег. Лайк, если хочешь слышать его чаще"
- **Эффект:** Раскачивает охваты, создает образ доброго эксперта

### Афоризмы от эксперта
```
Как [профессия] официально заявляю:
[Смелое/Поддерживающее утверждение]
```

### Астрология / Нумерология (мягкие ниши)
```
[Категория людей по дате/знаку]
[Что их ждет / Какие они]
[Если не нашли свое -> закреп / Telegram]
```
**Фишка:** Тех, чьих чисел нет -- переводить в Telegram.

### Метафорические карты (мягкие ниши)
```
Выбери карту [описание]
[Варианты в комментариях или -> Telegram]
```
**Важно:** НЕ выдавайте все карты! Переводите в Telegram.

## Где ставить ссылку

| Место | Можно? | Результат |
|-------|--------|-----------|
| В посте | НЕТ | Режется алгоритмами |
| В закрепленном комментарии | ДА | Оптимально |
| В шапке профиля | ДА | Обязательно |
| В обычном комментарии | ДА | Хуже закрепа |

**Важно:** Только ОТКРЫТЫЕ ссылки на Telegram! Закрытые не открываются с телефона.

## Твердые vs Мягкие ниши

### Твердые (маркетологи, SMM, продюсеры)

| Работает | Не работает |
|----------|-------------|
| Результаты с цифрами | Общая польза |
| Эмоциональные кейсы | Провокации (хейт) |
| Внутренний конфликт | Шаблонные посты |
| Лайфхаки | Копирование |

**Нюанс:** Нет шаблона -- нужно думать самому. Посты быстро выгорают.

### Мягкие (психологи, эзотерики, коучи)

| Работает | Особенности |
|----------|-------------|
| Вопросы на широкую | Раскачка за 5 дней |
| Лайк-посты | Каждый день |
| Астрология/нумерология | Золотая жила |
| Метафорические карты | Супер вовлечение |
| Афоризмы | Легко масштабировать |

**Нюанс:** Можно использовать шаблоны, повторять -- выгорают медленнее.

```
Утро: Лайк-пост / Пожелание

```

## 6 принципов виральности (STEPPS, Йона Бергер)

Перед публикацией -- минимум 2 из 6:

### 1. Социальная валюта
Люди делятся тем, что делает их круче. "Мало кто знает, но..." -- формат секретного знания.

### 2. Триггеры
Привяжи к ежедневному действию ЦА. "Каждый раз когда открываешь ChatGPT..."

### 3. Эмоции
ТОЛЬКО с высокой активностью: гнев, восторг, смех, шок. Спокойствие и грусть НЕ работают.

### 4. Общество
"Куда все, туда и я." Скрины с цифрами, кейсы клиентов, "327 человек уже забрали гайд".

### 5. Истории
Факты забываются, истории запоминаются. Снаружи байка -- внутри оффер.

### 6. Практическая ценность
Если реально экономит время/деньги -- сохранят и перешлют.

### STEPPS-фильтр (быстрая проверка)
- [ ] Социальная валюта -- репостнут чтобы выглядеть умным?
- [ ] Триггер -- привязан к повседневному действию?
- [ ] Эмоция -- бьет в нерв?
- [ ] Общество -- есть соц.доказательства?
- [ ] История -- оффер упакован в байку?
- [ ] Польза -- хочется сохранить/переслать?

## Threads Anti-patterns (платформенные)

| Ошибка | Как исправить |
|--------|---------------|
| Ссылка в посте | Только в закреп-коммент |
| Провокации/хейт | Добрые посты работают лучше |
| Нытье | Истории преодоления |
| Пиар под чужими постами | Давай ценность в комментах |
| Заходить расстроенным | Остынь, потом пиши |

Универсальные антипаттерны текста: см. text-posts (post-anatomy.md).
